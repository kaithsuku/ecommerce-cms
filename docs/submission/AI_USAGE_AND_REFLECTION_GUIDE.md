## Reflection Questions

Answer these in first person:

1. Which AI tools did I use and why?
2. Which parts of the system were AI-assisted?
3. Which parts did I manually review, test, or modify?
4. Did AI help me understand React, Express, Supabase, or integration better?
5. What problems did I face when using AI-generated code?
6. What did I learn from debugging and testing the application?

## Suggested Reflection Structure

```text
Paragraph 1: What I built and what AI tools I used.
Paragraph 2: How AI helped with planning, code, debugging, and documentation.
Paragraph 3: What I manually checked or changed.
Paragraph 4: Issues faced during integration and how I resolved them.
Paragraph 5: What I learned and what I would improve next.
```

## Important Academic Honesty Note

Do not submit generated reflection text unchanged. The reflection should describe
