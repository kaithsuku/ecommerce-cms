# Project Documentation

This folder contains the documentation required for the FSAD ecommerce CMS
assignment.

## Documents

- [API Documentation](API.md): REST endpoints, authentication, request bodies, responses, and error format.
- [Database Schema](DATABASE_SCHEMA.md): Supabase PostgreSQL tables, relationships, ER diagram, and seed SQL.
- [Architecture](ARCHITECTURE.md): System overview, application layers, request flow, and deployment/runtime notes.
- [Component Hierarchy](COMPONENT_HIERARCHY.md): React component tree, responsibilities, and frontend data flow.
- [UI/UX Wireframes](UI_UX_WIREFRAMES.md): Screen-level wireframes and interaction notes.
- [Assumptions](ASSUMPTIONS.md): Project scope, technical assumptions, limitations, and future enhancements.
- [Demo Script](DEMO_SCRIPT.md): Suggested video walkthrough.
- [AI Usage Log Template](AI_USAGE_LOG_TEMPLATE.md): Template for recording AI-assisted development.

## Submission Checklist

- Public GitHub repository with frontend, backend, and docs.
- Supabase database configured using the schema SQL.
- Demo video uploaded to Google Drive with evaluator access.
- AI usage log and reflection completed by the student.
