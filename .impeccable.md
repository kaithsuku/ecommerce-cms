## Design Context

### Users
Small-store admins and operations staff use this CMS while updating product content,
stock, categories, and order status. They need speed, clarity, and confidence more
than decorative ecommerce marketing.

### Brand Personality
Practical, composed, commerce-focused. The interface should feel like a modern
dark ecommerce admin dashboard: operational, polished, and trustworthy without
marketing-style drama.

### Aesthetic Direction
Shopify-inspired dark admin interface without copying Shopify. Use graphite
backgrounds, slightly raised panels, subtle borders, compact tables, calm
typography, and restrained green/teal action accents. Avoid glassmorphism, neon
gradients, oversized hero typography, cartoon rounding, and bare black-and-white
minimalism.

### Design Principles
- Make product and order operations scannable before they are decorative.
- Use panel grouping, type scale, and spacing as the primary visual tools.
- Keep tables, dialogs, filters, and forms compact but readable.
- Make every visible control directly useful to the admin workflow.
- Preserve accessibility with clear focus states, readable contrast, and reduced motion support.
